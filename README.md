# Decision Assistant / 辅助决策工具

一个基于 HTML、CSS、JavaScript 的本地辅助决策工具，用于帮助用户在多个方案之间进行结构化比较。

项目支持问题记录、评估维度设置、方案打分、加权排序、行动清单、自由笔记，以及 JSON 导入导出。

## 功能特点

- 新建多个决策项目
- 记录问题背景、目标和约束
- 添加评估维度并设置权重
- 添加多个备选方案
- 为不同方案按维度打分
- 自动计算加权得分并排序
- 添加行动清单和自由笔记
- 使用 LocalStorage 在浏览器本地保存数据
- 支持 JSON 导入和导出

## 技术栈

- HTML
- CSS
- JavaScript
- LocalStorage
- JSON
